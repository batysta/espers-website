Frequently Asked Questions

How to use Espers Wallet

How do i install a wallet (Windows/Mac)

WINDOWS:
Download the file, and unpack the launcher to your desktop or any folder.
Run the espers launcher, it should do some installation first, after that, close the launcher, and reopen it. Now hit “Check for updates” and if it says, you are on latest version, and you can run espers wallet from the launcher.


MAC:
Unpack the files to your desktop or folder.
Right click on Espers-qt.app to open the menu, hold down option key, and click open (ONLY WHEN YOU RUN THE WALLET FIRST TIME)

How do i install a wallet (Ubuntu)

You can locate the daemon afterwards in at /Espers/src/Espersd

Synchronization is slow / I have a low amount of active connections.

If you're having a low amount of active connections, try adding nodes from the nodes list to your configuration file. A list of current nodes can be found here.

On Windows, the configuration file can be found at C:\Users\(your-username)\AppData\Roaming\ESP\Espers.conf.

On Mac OS it's located at /Users/(your-username)/Library/Application Support/ESP/Espers.conf.

How do i stake my coins

Your Epsers needs to mature in your wallet approx 2 hours before they can start staking.
You can have your wallet encrypted, but it needs to be unlocked. You can do this from settings. Wallet needs to stay open and connected.

How to mine Espers

HOW TO MINE WITH YOUR CPU

Download a miner from
http://cryptomining-blog.com/7805-new-cpuminer-multi-fork-with-hmq1725-algorithm-support/

Extract the files to a folder on your desktop, and locate the RUN-AMD or RUN-INTEL file, (CHOOSE WICH CPU YOU HAVE)
Right click the chosen file, and edit it
Change after: -a HMQ1725 -o stratum+tcp://stratum.minertopia.org:3747 -u YOUR ESPERS WALLET ADDRESS -p c=ESP -t AMOUNT OF THREADS YOU WANT TO USE
If you want to utilize your whole CPU, remove the -t part.
Save and exit, and run the .bat file you chosen (based on wich brand CPU you have)
Enjoy, you are now mining!

HOW TO MINE WITH YOUR GPU (NVIDIA)

Download CCminer from
https://github.com/tpruvot/ccminer/releases

Extract the files to a folder on your desktop
Choose one .bat file, right click it and hit edit.
it should look like this

ccminer-x64 -a HMQ1725 -o stratum+tcp://stratum.minertopia.org:3747 -u YOUR ESPERS WALLET ADDRESS -p c=ESP pause

Save and exit, and run the .bat file

HOW TO MINE WITH YOUR GPU (AMD)

Download SGminer with HMQ1725 algo from
http://cryptomining-blog.com/tag/sgminer-hmq1725/

Extract the files to a folder on your desktop
Choose one .bat file, right click it and hit edit.
it should look like this

setx GPU_FORCE_64BIT_PTR 0
setx GPU_MAX_HEAP_SIZE 100
setx GPU_USE_SYNC_OBJECTS 1
setx GPU_MAX_ALLOC_PERCENT 100

sgminer.exe –no-submit-stale –kernel hmq1725 -o stratum+tcp://stratum.minertopia.org:3747 -u wallet -p x -I 18

Save and exit, and run the .bat file
